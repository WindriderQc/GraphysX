# Three.js Hello World

A simple Three.js scene featuring a red bouncing ball on a grey plane, built with Vite.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open your browser to the URL shown in the terminal (usually `http://localhost:5173`).

### Building for Production
1. Generate the production build:
   ```bash
   npm run build
   ```
2. The optimized files will be in the `dist` directory.

### Deployment
The contents of the `dist` folder can be hosted on any static hosting service like Netlify, Vercel, or GitHub Pages.
